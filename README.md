# book-studies

This repository holds a collection of book studies designed to help small groups of 4-6 individuals work through books together. The goal is to foster learning and build connections among participants while exploring various topics.

## About Book Studies

Book studies in this repository provide a structured approach for small groups to:
- **Learn together** - Dive deep into topics through guided discussion and shared insights
- **Connect with others** - Build relationships and get to know fellow participants
- **Explore diverse topics** - Choose from technical or non-technical books based on group interests

Each book study includes materials and structure to facilitate meaningful group discussions and collaborative learning.
